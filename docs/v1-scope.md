# V1 Scope

Mneme v1 is ready when it is a safe container for capturing and reviewing
durable personal context.

## Included

- Record Contract documented.
- Folder structure documented and present.
- Inbox Workflow documented.
- ChatGPT Intake prompt documented.
- Digest format documented.
- Graph policy documented.
- Rust validator implemented.
- Pre-commit hook available.
- Local-only folder skeleton present in fresh clones.

## Deferred

- Digest generator implementation.
- Graph generator implementation.
- Typed graph relationships.
- Encrypted backup and restore workflow.
- Real Durable Records.

V1 does not require real Durable Records. It provides the structure needed to
start dumping knowledge into `inbox/` and reviewing promoted records through
pull requests.
